# Raw Inbox

Use this folder as a temporary intake queue for material that has not yet been triaged.

Examples:

- source metadata exports;
- reading-list items;
- short meeting notes;
- project-decision notes;
- draft source summaries;
- requirement pointers.

Before moving any item onward:

1. remove private or restricted content;
2. record the source;
3. assign an evidence status;
4. add a row to `../growth-queue.md`;
5. decide the canonical destination.

Do not store raw participant data, signed consent forms, recordings, restricted LMS content, credentials, or private client records here.
