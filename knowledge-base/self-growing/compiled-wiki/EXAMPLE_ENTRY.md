## What This Node Is For

Use this page to show how a compiled wiki entry should work.

A compiled wiki node should connect source notes, project decisions, and open questions. It should not become the only place where evidence is stored.

## What It Can Support

- finding relevant source notes;
- summarising a recurring theme;
- showing which files need to be checked before writing;
- identifying unresolved questions.

## What It Cannot Support Yet

- formal citation claims without source-section review;
- official requirements without checking the requirement source;
- participant, client, or institutional claims without compliance evidence.
