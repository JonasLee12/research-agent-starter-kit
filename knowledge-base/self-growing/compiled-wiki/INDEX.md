## How To Use This Index

1. Start here to locate the right canonical file.
2. Open the source-of-record file before making formal claims.
3. Use retrieval only to find candidate files, not to prove a claim.
4. Add new nodes only when they connect at least two source-of-record files or clarify a recurring project decision.
