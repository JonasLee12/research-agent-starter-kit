# Source Readiness Matrix

Purpose: track whether a source can be used for formal writing.

## Status Labels

| Status | Meaning |
|---|---|
| `METADATA ONLY` | Bibliographic details only. Do not use for claims. |
| `ABSTRACT REVIEWED` | Abstract has been checked. Useful for screening, not strong claim support. |
| `TARGETED REVIEWED` | Relevant sections have been reviewed for a specific claim or section. |
| `FULL TEXT REVIEWED` | Full source has been read or closely reviewed. |
| `CLAIM SUPPORT VERIFIED` | A specific claim has been checked against the relevant source section. |
| `TO CONFIRM` | Evidence status is unclear. |

## Template

| Source ID | Citation | Current Status | Can Support | Cannot Support Yet | Notes |
|---|---|---|---|---|---|
| `SOURCE_ID` | `TO CONFIRM` | `METADATA ONLY` | `TO CONFIRM` | formal claims | Add source note before upgrading readiness. |

## Rule

Metadata is not evidence. Upgrade a source only after reading the relevant section.
