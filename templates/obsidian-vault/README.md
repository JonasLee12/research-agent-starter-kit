# Clean Obsidian Vault Template

Copy this whole folder to a location outside this repository, rename the copy for your own project, and open the copied folder in Obsidian.

Do not open the repository root as an Obsidian vault. The repository root contains agent system files, scripts, tests, audit logs, and templates that will pollute the graph.

This template is intentionally generic. It contains no private project content, no institution names, no supervisor details, no email addresses, no participant material, no screenshots, and no private documents.

Suggested use:

- `00_Home/` for navigation.
- `01_Project/` for project overview and scope.
- `02_Proposal/` for proposal planning.
- `03_Literature/` for reading notes and literature clusters.
- `04_Methodology/` for methods and design decisions.
- `05_Ethics/` for non-sensitive ethics planning.
- `06_Design_or_Output/` for design outputs, models, figures, or deliverables.
- `07_Tasks/` for task lists and weekly plans.
- `08_Templates/` for reusable note templates.
- `09_Private_Data_DO_NOT_SYNC/` for local reminders only. Do not push private contents to GitHub.
- `Attachments/` for local non-sensitive attachments.

Obsidian is a navigation layer. Keep source-of-record evidence in your project source register, source notes, and reviewed source files.
