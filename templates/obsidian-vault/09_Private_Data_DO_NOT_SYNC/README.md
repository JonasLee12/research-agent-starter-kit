# Private Data Do Not Sync

This folder is a local reminder only.

Use it only in your copied personal vault for private material that should not be committed, pushed, shared, or synced without review.

Examples of material that requires extra care:

- participant details;
- interview transcripts or recordings;
- consent forms or signatures;
- private institutional guidance;
- assessment drafts;
- screenshots from private systems;
- local paths, tokens, credentials, or browser/session data.

Do not add private contents from this folder to GitHub.
