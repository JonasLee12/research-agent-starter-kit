# Public Release Renaming Guide

Before making this repository public, use generic names.

## Recommended Repository Names

Good options:

- `research-agent-starter-kit`
- `research-project-agent-template`
- `research-workflow-agent-kit`
- `academic-writing-agent-workbench`
- `evidence-synthesis-agent-template`

Avoid names that include:

- a real student name;
- a university name;
- a supervisor, PI, client, reviewer, or funder name;
- a current private project title;
- `private` if the repository will become public;
- a module code that only makes sense at one institution.

## Recommended Local Folder Names

```text
~/Documents/research-agent-starter-kit
~/Documents/my-research-agent
~/Documents/project-research-copilot
```

## GitHub Visibility

Keep the repository private until:

- `scripts/privacy_check.sh` passes;
- `PUBLIC_RELEASE_AUDIT.md` is complete;
- a manual search finds no private project content;
- licence and contribution rules are chosen.
- GitHub Actions privacy check passes after the latest push.
