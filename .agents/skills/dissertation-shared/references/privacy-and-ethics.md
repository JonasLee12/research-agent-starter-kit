# Privacy and Ethics Protocol

This dissertation may involve university teachers, interviews, workshops, surveys, or co-design materials.

Rules:

- Treat raw transcripts, names, departments, institutions, email addresses, and identifiable teaching scenarios as sensitive.
- If raw data is present, use participant IDs or pseudonyms only.
- Do not copy sensitive raw excerpts into outputs unless the user explicitly asks and the excerpt is already anonymized.
- Flag accidental identifiers before using material in chapters, slides, or appendices.
- Do not infer demographic attributes not present in the data.
- Distinguish participant evidence from the researcher's interpretation.
- If the user asks for sharing, publication, or external tool use with sensitive data, ask for confirmation first.
