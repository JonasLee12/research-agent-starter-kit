# Output Contracts

Recommended project files:

- `research-wiki/`: project memory and notes.
- `research-wiki/LITERATURE_MAP.md`: literature clusters and gaps.
- `research-wiki/METHOD_DECISIONS.md`: design decisions and rationale.
- `research-wiki/SUPERVISOR_FEEDBACK.md`: feedback and response history.
- `chapter-plans/`: chapter outlines and argument maps.
- `audit-reports/`: citation, theme, and methodology audits.
- `figures/`: figure specifications and diagrams.
- `viva/`: defense questions, answers, and talk notes.

For any audit or review, include:

- date
- files reviewed
- confirmed findings
- uncertain points
- action list
