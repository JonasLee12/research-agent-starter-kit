# Citation Discipline

Rules:

- Do not fabricate references, DOIs, page numbers, quotes, or publication metadata.
- Mark unverified references as `NEEDS VERIFICATION`.
- A citation must support the exact claim attached to it, not just the general topic.
- Prefer primary sources for theories, frameworks, and methodological concepts.
- For contested claims, include multiple sources or state the uncertainty.
- When auditing, record:
  - citation key or author-year
  - claimed support
  - evidence location
  - status: verified, weak support, mismatch, missing, or needs verification
