These PaperSpine-inspired components are intentionally not active in the current dissertation workflow. Revisit them later when the project reaches the relevant stage.

## Defer Until Publication Or Journal Targeting

- target journal/conference research,
- recent venue paper style survey,
- article-type requirements,
- cover-letter or journal-fit logic.

Reason: the current task is a PROJECT_UNIVERSITY dissertation proposal/ethics-stage project, not a journal submission.

## Defer Until Full Draft Exists

- full style imitation from 3-6 exemplar papers,
- paragraph-level style metrics,
- automated shallow-edit revision audit.

Reason: there is not yet a complete dissertation manuscript to compare or rewrite.

Read `paperspine-adapted-writing-controls.md` for the adapted versions. They are designed for proposal, chapter, supervisor-feedback, and Word-based dissertation work, not for the full PaperSpine journal/LaTeX pipeline.

Reason: current value comes from the writing workflow, not from running repository scripts.

Consider migrating more PaperSpine components when:
