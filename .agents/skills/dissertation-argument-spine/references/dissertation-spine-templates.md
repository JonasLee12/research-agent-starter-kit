## Seven-Anchor Dissertation Test

Extract these seven sentences or planned sentence roles:

1. title or subtitle promise,
2. first Introduction problem sentence,
3. main gap sentence,
4. aim sentence,
5. methodology rationale sentence,
6. expected contribution sentence,
7. conclusion/discussion implication sentence.

They should tell one coherent story. If they do not, revise the section blueprint before polishing language.
